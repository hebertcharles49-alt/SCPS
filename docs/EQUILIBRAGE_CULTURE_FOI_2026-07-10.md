# SCPS : équilibrage des éthos, héritages, traditions et foi (spec joueur, 2026-07-10)

> Audit fourni par le joueur, verbatim opérationnel. À appliquer APRÈS l'atterrissage du
> câblage traditions↔registre J (l'agent en cours change ce que « vivant » veut dire —
> les recommandations Arcanique/Sourd à l'arcane notamment supposaient arcane = flux seul).

## Unité commune (traditions)
Mineure ≈ 10 % démo/productivité, OU 0,5 K/P/H/fracture, OU 0,25-0,5 influence.
Majeure = 2 mineures. Défaut = 1 mineure (validation exige 1 majeur + 1 mineur + 1 défaut).

## ÉTHOS
- DOMINATEUR (Horde) : garder orientation militaire + tolérance élite basse ; RETIRER
  l'esclavage sans tech (exiger Économie servile — ⚠ revert partiel du lot brassage,
  décision joueur) ; signature d'héritage normalisée. Flavor : « Ils ne demandent pas si
  la frontière peut être franchie, seulement combien d'hommes il faudra pour qu'elle
  cesse d'exister. »
- HONNEUR (Clans) : esclavage → tech ; identité razzia/prestige ; tolérance élite
  0,22→0,30-0,32. Flavor : « Une dette peut être oubliée, une défaite réparée. Une honte,
  elle, attend patiemment les petits-fils. »
- ORDRE (Ordre) : quasi intact — le POINT DE RÉFÉRENCE ; tolérances cibles 0,50/0,50/0,45.
  Flavor : « Chaque personne connaît sa place, chaque place son devoir et chaque devoir le
  sceau qui le rend incontestable. »
- BUREAUCRATE (Couronne) : tolérances 0,60/0,60/0,58 → 0,55/0,52/0,50 ; biais
  institutionnel 1,6 → 1,35 ; garder l'intégration comme identité. Flavor : « Le royaume
  ne repose pas sur la volonté d'un seul homme, mais sur mille registres qui refusent
  obstinément de se contredire. »
- MERCANTILE (Ligue) : garder tolérance bourgeoise basse ; meilleure PORTÉE commerciale
  (pas une meilleure stabilité). Flavor : « Ils ne conquièrent pas les ports. Ils y
  prêtent de l'or jusqu'à ce que les clés deviennent une modalité de remboursement. »
- PACIFISTE (Havre) : tolérance uniforme 0,30 → 0,38 ; surcoût militaire IA 1,30 → 1,20 ;
  garder frein expansion/faustien. Flavor : « Ils ont juré de ne prendre aucune vie. Leurs
  voisins débattent encore pour savoir si cette promesse est une vertu ou une invitation. »
